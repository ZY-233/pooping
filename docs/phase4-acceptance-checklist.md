# 阶段 4 人工验收 Checklist（可执行）

## 0. 预备条件
- [ ] `.env.local` 已配置：`NEXT_PUBLIC_SUPABASE_URL`、`NEXT_PUBLIC_SUPABASE_ANON_KEY`
- [ ] Supabase 已执行 `supabase/migrations/20260313190000_phase2_init.sql`
- [ ] Supabase Auth 已开启 `Anonymous sign-ins`
- [ ] 本地启动成功：`npm run dev`

## 1. 首页 -> 创建记录
- [ ] 打开 `http://127.0.0.1:3000/`
- [ ] 看到标题“首页”和按钮“记录一次”
- [ ] 点击“记录一次”打开弹层
- [ ] 选择：形状=“正常”，感受=“很顺”，备注输入 `manual-<当前时间戳>`
- [ ] 点击“保存记录”
- [ ] 页面出现“记录好了，今日顺顺 +1”

## 2. 首页摘要与最近记录
- [ ] 本周次数 / 本月次数有变化（至少 +1）
- [ ] “最近一次记录”时间更新为刚创建时间
- [ ] “最近记录”列表出现新条目，可点击

## 3. 记录详情 -> 编辑 -> 删除
- [ ] 点击最近记录进入 `/records/[id]`
- [ ] 详情页展示时间、形状、感受、备注
- [ ] 点击“编辑记录”，修改备注为 `manual-edit-<当前时间戳>`，保存
- [ ] 页面出现“记录已更新”，且详情备注变为新值
- [ ] 点击“删除记录”并确认
- [ ] 自动返回首页
- [ ] 重新访问刚才的 `/records/[id]`，出现“这条记录不存在，或已被删除。”

## 4. 日历与回看闭环
- [ ] 打开 `/calendar`
- [ ] 看到“本月一共记录 N 次”
- [ ] 点击“下一月”后标题变化，再点“上一月”恢复
- [ ] 当前月有记录日期高亮 + 圆点
- [ ] 选中某一日期后，下方出现“当天记录”列表
- [ ] 点击列表项可跳转到 `/records/[id]` 详情

## 5. 非目标确认（阶段 4 边界）
- [ ] 没有复杂图表
- [ ] 没有趋势分析
- [ ] 没有 AI 总结
